CTI - Chamados TI

Chamados TI é um sistema em PHP usado como alerta do setor de TI para ambientes educasionais.
O sistema fica hospedado em uma máquina virtual rodando com um banco de dados local no XAMPP.
O professor acessa o sistema por um ícone na área de trabalho e responde o formulário, após abrir o chamado a equipe de TI receberá a notificação em sua sala quando a tela ficar vermelha.
O técnico responsável precisa logar no sistema para realizar a finalização do chamado.

Sistema feito utilizando HTML5, Bootstrap, MySQL e PHP.
O código fonte do sistema pode ser acessado via Github.
